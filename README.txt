1. Atidarome setup.exe file'ą ir įdiegiame jį į kompiuterį.
2. Po sėkmingo įdiegimo galime paleisti programą.
3. SVARBU - programa turi būti paleista per Run As Administrator, kitaip išmes klaidą rušiuojant studentus.
4. Norint nustatyti, kad visada desktop shortcut klaustu dėl administrator teisių, turime užeiti į ant shortcut - > Properties -> Compatibility - > Uždedame varnelę ant Run this program as an administrator - > Apply - > OK
5. Naudokite programą naudodamiesi valdikliais ir juose nurodyta informacija.

Programa gali dirbti su txt file'ais.

TXT failo eilutės formatas : 

Vardas Pavardė ND_pažymiai Egzamino_pažymys

Pvz :.

Vardenis Pavardenis 10 6 7 6 7 4 3 6

Gero naudojimo!
